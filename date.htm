Date

Data Type

fontes: w3school ref, w3school, devdocs.oi MDN

new Date() - possui x assinaturas, que retornam object

var d = new Date(2017, 5);  //--retorna o 1º dia do mês de Junho
var d2 = new Date(2017, 5, 2, 25);     //--retorna o dia 3 de Junho às 1:00
var d3 = new Date(2017, 5, 2, 10, 61); //--retorna o dia 2 de Junho as 11:01
var d4 = new Date(2017, 5, 2, 10, 10, 5, 5); //---assinatura vai ate o milisegundo

Propriedade e métodos
getFullYear() - ano
getMonth() - mês (0 janeiro)
getDate() - dia
getDay() - dia da semana (0 domingo)
getHours() - hora
getUTCHours() - hora sem fuso horário (Universal Coordinated Time (UTC)), Brasília +3
getMinutes(), getSeconds()
getTime() - milisegundos desde 1970

toLocaleString() - data com string, hora local (Brasília) yyyy-mm-dd hh:mm:ss
toISOString() - data com string, sem sem fuso horário yyyy-mm-ddThh:mm:ss.mmmZ - toJSON()
toLocaleDateString() - yyy-m-dd
toLocaleTimeString() - hh:mm:ss
toString(), toTimeString()

now(), parse() - statics Date.now(); Date.parse()
